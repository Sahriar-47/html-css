My Simple Webpage

Welcome to my simple webpage! This simple webpage demonstrates my beginner-level HTML and CSS skills. It includes a scrolling marquee to show welcome messages and highlights about my projects. I integrated MathJax to render clean mathematical equations directly in the browser. The page also embeds an audio player for sound, a YouTube video using an iframe, and a PDF viewer to display documents. I focused on clean, simple design with readable fonts and spacing, using a separate CSS file to style elements. This project helped me understand linking CSS externally, embedding media, and structuring HTML for clarity and usability.
